Consumer360 – Retail Customer Analytics Project

Project Overview
This project analyzes retail transaction data to identify high-value customers,
detect churn risk, and discover purchasing behavior using RFM analysis.

Features
- RFM Segmentation (Recency, Frequency, Monetary)
- Cohort Analysis for customer retention
- Market Basket Analysis for product association
- Automated Data Pipeline
- Ready for Power BI Dashboard Integration

Tech Stack
- SQL (Data Extraction)
- Python (Pandas, NumPy)
- Power BI (Visualization)
